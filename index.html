About Me

I am a technical project manager with a passion for creating. I am also a self-proclaimed nerd. I have always enjoyed working in technology for its dynamic, ever-evolving landscape. Most recently, I was a project manager for a small privately owned tech consulting company where I created client specific landing pages and helped refactor the company website. As a technical project manager I managed many complex projects with a talented team of UX/UI designers and developers. We created everything from HMIs for industrial machines, to GUIs for remote monitoring devices, to user experience flows for client onboarding applications.

I am energized by environments in which I can combine creativity and problem solving!



Projects

Six projects I enjoyed working on are highlighted below. Select a project image for more information, to view the deployed project or to see the code behind it all! Visit my full GitHub repository at: Onomatopoetica. Note: Apps deployed on Heroku take a few seconds to load.


  • EVOLVE FITNESS JOURNAL


    A fitness journal using a Node + Express server with a mySQL database + a Sequelize ORM following an MVC design paradigm. A full stack application deployed on Heroku.


  • EAT DA BURGER


    A burger logger to stage, devour + delete burgers using mySQL, Node, Express, Handlebars + a homemade ORM (yum!). An MVC design paradigm was followed. Deployed on Heroku.

  • NOTE TAKER


    This application utilizes an Express backend. Uniqid npm generates an id for each note. The app saves, retrieves, and deletes note data from a JSON file. It is deployed on Heroku.

  • PICKY EATER


    A group project using jQuery + Spoonacular API to create a recipe resource for people with food sensitivities to search recipes matching their dietary restrictions.

  • CODE QUIZ


    This quiz app gives students 75 seconds to take a JavaScript code quiz testing their knowledge of the fundamentals. The app stores high scores enabling students to gauge their progress compared to their peers. This application is deployed on GitHub.

  • TEAM PROFILE GENERATOR


    A Node CLI takes in information about employees + dynamically generates an HTML profile displying summaries for each person. Jest runs testing + Inquirer npm collects user input. The app runs in Terminal and cannot be deployed to GitHub.
